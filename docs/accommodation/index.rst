Система проживания
=====================

Этот блок используется для автоматизации размещения участников мероприятия по помещениям и/или домам: создается связь между заявкой и местом в конкретном помещении, автоматически добавляется цена за это место к требуемому оргвзносу. Система проживания в первую очередь предназначена для конвентов, но может быть полезна играм на базах.
